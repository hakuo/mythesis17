1. OpenCV installation
    - Install dependencies:
        sudo apt-get install build-essential cmake git libgtk2.0-dev pkg-config libavcodec-dev libavformat-dev libswscale-dev python-dev python-numpy libtbb2 libtbb-dev libjpeg-dev libpng-dev libtiff-dev libjasper-dev libdc1394-22-dev

    - Download source: https://opencv.org/releases.html
    - Or clone source from git:
        git clone https://github.com/opencv/opencv.git
        git clone https://github.com/opencv/opencv_contrib.git
        cd opencv && mkdir build && cd build
        cmake -D CMAKE_BUILD_TYPE=Release -D CMAKE_INSTALL_PREFIX=/usr/local ..
        make -j7                                                                # runs 7 jobs in parallel
        sudo make install

    - Find opencv library path:
        pkg-config opencv --libs

    - Create and insert opencv library path in it

    - Active library:
        sudo ldconfig -v

    - Find opencv include path: pkg-config opencv --cflags.

    - Find opencv library path: pkg-config opencv --libs.

    - Open Project > Properties > C/C++ Build > Setting.

    - Add include path to Include paths (-I): GCC C++ Compiler > Includes.
        E.g.: /usr/local/include/opencv

    - Add library path to Library search path (-L): GCC C++ Linker > Libraries.
        E.g.: /usr/local/lib

    - Add libraries you need to Libraries (-l): GCC C++ Linker > Libraries. 
        E.g.: opencv_core, opencv_imgproc, opencv_highgui, opencv_ml, opencv_imgcodecs, opencv_dnn

2. Leptonica installation
    git clone https://github.com/DanBloomberg/leptonica.git
    cd leptonica
    mkdir build
    cd build
    cmake .. -DBUILD_PROG=1         # To make both the library and the programs
    make
    sudo make install

3. Tesseract installation (online guide: https://github.com/tesseract-ocr/tesseract/wiki/Compiling)
    - Install dependencies
    sudo apt-get install g++ autoconf automake libtool autoconf-archive pkg-config libpng12-dev libjpeg8-dev libtiff5-dev zlib1g-dev

    - Install training tools
    sudo apt-get install libicu-dev libpango1.0-dev libcairo2-dev

    - Clone source from git and build
    git clone https://github.com/tesseract-ocr/tesseract.git
    cd tesseract
    ./autogen.sh
    ./configure --enable-debug
    make
    sudo make install
    make training
    sudo make training-install

4. Setting up project
    - Project > Properties > C/C++ Build > Settings > GCC C++ Compiler > Miscellaneous > Other flags:
        -std=c++11 -DUSE_STD_NAMESPACE

    - Project > Properties > C/C++ Build > Settings > GCC C++ Linker > Libraries > Libraries (-l):
        lept
        tesseract
    - Before run executable file: 
        export TESSDATA_PREFIX=/usr/share/tesseract-ocr/tessdata
        ./Sample_OCR scenetext01.jpg
